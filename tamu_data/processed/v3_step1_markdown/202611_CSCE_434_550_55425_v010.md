[16.0pt] Catalog Description
Compiler Design. (3-0). Credit 3. Programming language translation; functions and
general organization of compiler design and interpreters; theoretical and
implementation aspects of lexical scanners; parsing of context free languages; code
generation and optimization; error recovery. Prerequisite: Grade of C or better in
CSCE 315 or CSCE 331.
[16.0pt] Additional Course Details
In this course the students will develop a fundamental understanding of the issues
that arise in program translation, including lexing, parsing, syntax analysis,
translation, register allocation, code generation and rudimentary program
optimization. The programming assignments will help students develop skills in the
creation and manipulation of abstract program representations and implementation
of complex software algorithms. This will provide deeper insights into the more
advanced semantics aspects of
programming languages, code generation, machine independent optimizations,
dynamic memory
allocation, types and their inferences and object orientation. Exam questions will be
drawn from both the book and the lectures.
[16.0pt] Course Prerequisites
Prerequisite/Corequisite(s):  Grade of C or better in CSCE 315 or CSCE 331.
[16.0pt] Course Learning Outcomes
Upon completion of this course, the learner will be able to:
Apply the translation process from user code to machine code and analyze the
functionality of each stage of the compilation process.
Create grammars for programming languages and apply parsing techniques to
detect syntactic and semantic errors.
Apply the theoretical foundations to use compiler construction tools efficiently
Analyze the complexities and tradeoffs in the compilation process while
applying optimizations to extract the maximum code performance.
Create an optimizing compiler that can generate executable machine code
from source code.

[13.0pt] No Use of Generative AI Tools Permitted
According to the Texas A&M University Definitions of Academic Misconduct, plagiarism is the
appropriation of another person's ideas, processes, results or words without giving
appropriate credit (aggiehonor.tamu.edu). You should credit your use of anyone else's words,
graphic images, or ideas using standard citation styles. Artificial Intelligence (AI) text
generators and natural language processing tools (colloquially, chatbots - such as ChatGPT),
audio, computer code, video, and image generators should not be used for any work for this
class without explicit permission of the instructor and appropriate attribution. This includes,
but is not limited to,
i. Creating or revising drafts
ii. Editing your work
iii. Reviewing a peer's work
This excludes pre-existing software additions such as spelling and grammar checkers, which
are acceptable.
All assignments should be fully prepared by the student. Developing strong competencies in
the skills associated with this course, from student-based brainstorming to project
development, will prepare you for success in your degree pathway and, ultimately, a
competitive career. Therefore, the use of generative AI tools to complete any aspect of
assignments for this course are not permitted and will be treated as plagiarism. If you have
questions about what constitutes a violation of this statement, please contact the instructor.

inaccurate and misleading data through informal and formal hearings.

items can do so within  howdy.tamu.edu  using the Directory Information

be made in this class to protect your privacy and to ensure confidential treatment
