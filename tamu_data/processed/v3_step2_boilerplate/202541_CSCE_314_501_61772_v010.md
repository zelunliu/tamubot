[16.0pt] Course Description
[10.5pt] Exploration of the design space of programming languages via an in-depth study of
[10.5pt] two programming languages, one functional and one object-oriented; focuses on
[10.5pt] idiomatic uses of each language and on features characteristic for each language.
[16.0pt] Course Prerequisites
Prerequisite/Corequisite(s):  Grade of C or better in CSCE 221, or concurrent
enrollment.
[16.0pt] Special Course Designation
None
[16.0pt] Course Learning Outcomes
Upon completion of this course, the learner will be able to:
Use modern programming languages more effectively to build correct and
elegant software • Develop the necessary skills to learn and adopt new
programming languages quickly
Understand language constructs, common abstraction mechanisms, and
efficiency considerations
Evaluate the power of programming languages and their constructs
Although the course uses and studies specific programming languages, learning
these languages should be considered as a side benefit of the course, not the main
goal.

[16.0pt] AI Statement
The use of AI to gain a greater understanding of a topic is a great use of its
capabilities. This is allowed.
The use of AI to do your assignments is a poor use of its capabilities, as it does not
increase your capabilities. This is highly discouraged. If done, you must cite the AI
tool you used and give the prompts and responses given/received.
There is no access to any form of AI during exams, and historically, those who used
AI on assignments did poorly on exams.
