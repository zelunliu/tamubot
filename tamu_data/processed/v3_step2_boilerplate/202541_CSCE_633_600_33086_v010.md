[16.0pt] Course Description
[12.0pt] Machine learning is a field of study that gives computers the ability to learn without
[12.0pt] being explicitly programmed. It is the core of AI. In the past decade, machine
[12.0pt] learning has created tremendous value with its application in various domains,
[12.0pt] including information retrieval, speech recognition, computer vision, understanding
[12.0pt] of human genome, natural language processing, etc. In this class, students will not
[12.0pt] only learn the fundamental machine learning techniques, but also gain the hands-on
[12.0pt] experience by applying these techniques and developing new techniques for solving
[12.0pt] problems from the real world. In addition, students will learn some recent advances in
[12.0pt] deep learning and AI. Topics of interest include: (1) Regression (least square
[12.0pt] regression, lasso). (2) Classification (support vector machines, logistic regression).
[12.0pt] (3) Kernel Methods. (4) Unsupervised methods; (5) Self-supervised learning; (6)
[12.0pt] Stochastic Optimization. (7) Deep Learning. By the end of class, students are
[12.0pt] expected to accomplish a project related to machine learning in a group.

[16.0pt] Additional Course Information
[13.0pt] Collaboration
[12.0pt] Both homework assignments and projects are team-based, and each student must join
[12.0pt] a team of 2–3 members to collaborate. Only one copy of the homework or project
[12.0pt] needs to be submitted per team, and every submission must clearly acknowledge each
[12.0pt] member’s contributions. If a student is found to be uncooperative, they may be
[12.0pt] removed from the team and required to form or join a new team if necessary.
