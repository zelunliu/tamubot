[16.0pt] Catalog Description
Database Systems. (3-0). Credit 3. File structures and access methods; database
modeling, design and user interface; components of database management
systems; information storage and retrieval, query languages, high-level language
interface with database systems. Prerequisites: CSCE 221 with a grade of C or
better; junior or senior classification.
[16.0pt] Course Prerequisites
Prerequisite/Corequisite(s):  CSCE 221 with a grade of C or better; junior or senior
classification.
[16.0pt] Course Learning Outcomes
Upon completion of this course, the learner will be able to:
• Explain the fundamental concepts of database systems, including data models,
database architecture, and the role of database management systems (DBMS)
• Design and implement Entity-Relationship (ER) and Enhanced Entity-Relationship
(EER) models to represent real-world data requirements
• Apply normalization theory and functional dependencies to design efficient and
well-structured relational database schemas
• Write complex SQL queries to create, manipulate, and retrieve data from
relational databases
• Evaluate and compare different database paradigms, including relational
databases, NoSQL systems, and distributed databases
• Analyze data warehouse architectures and understand their application in
business intelligence and decision support systems
• Implement database connectivity in applications and apply security principles to
protect database systems

[16.0pt] Course Schedule
Data modeling: Relational data model, database design, normalization
SQL: Create, select, insert, joins, subqueries, views, stored procedures, window
functions
NoSQL: Key-value (Redis), document (MongoDB), column (Cassandra), graph and
vector databases
Software development: JDBC, Object-Relational Mapping
Distributed databases: Replication and partitioning, sharding, CAP theorem
DBMS Internals: Storage and file organization; indexing: B+ trees, hash tables;
buffer and memory management; query processing and optimization
