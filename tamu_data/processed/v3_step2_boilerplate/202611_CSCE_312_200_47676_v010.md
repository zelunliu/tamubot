[16.0pt] Catalog Description
Computer Organization. (3-2). Credit 4. Computer systems from programmer's
perspective including simple logic design, data representation and processor
architecture, programming of processors, memory, control flow, input/output, and
performance measurements; hands-on lab assignments. Prerequisite: Grade of C
or better in CSCE 221, or concurrent enrollment.
[16.0pt] Course Prerequisites
Prerequisite/Corequisite(s):  Grade of C or better in CSCE 221, or concurrent
enrollment.
[16.0pt] Course Learning Outcomes
Upon completion of this course, the learner will be able to:
Perform Basic Logic Design
Understand the design of a simple computer system
Understand elements that make up a CPU (Central Processing Unit)
Design software with an understanding of how it will run on computers
Participate in discussions that involve impact of hardware and software
elements on each other and the performance of the program
Take intermediate and advanced classes in computer architecture,
systems, operating systems, cloud, parallel computing
Program for performance, including efficiently using the cache hierarchy

[16.0pt] AI Statement
[12.0pt] AI is so good these days that I can’t tell if you’re using it. If I tell you not to
[12.0pt] use it, it makes it difficult for you because you suspect your classmates will
[12.0pt] do it anyway, so why shouldn’t you do it too if you want to get a good grade?
[12.0pt] Rather, I try to reduce the incentives to use it. So the homeworks are worth
[12.0pt] only 10%, while the exams are worth 70%, and you can’t use AI on the
[12.0pt] exams because I don’t allow devices in the exams. If you use AI to do the
[12.0pt] homework for you, you might get that full 10% for the homeworks but you
[12.0pt] won’t learn what you need to learn to pass the exams so you’ll fail anyway. If
[12.0pt] you get stuck on a homework assignment, feel free to consult AI to help you
[12.0pt] as you would a teaching assistant or instructor, but make sure you’re
[12.0pt] actually learning the material rather than letting the AI just do it for you.
