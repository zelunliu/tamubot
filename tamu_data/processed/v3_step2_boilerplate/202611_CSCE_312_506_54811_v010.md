[16.0pt] Catalog Description
Computer Organization. (3-2). Credit 4. Computer systems from programmer's
perspective including simple logic design, data representation and processor
architecture, programming of processors, memory, control flow, input/output, and
performance measurements; hands-on lab assignments. Prerequisite: Grade of C
or better in CSCE 221, or concurrent enrollment.

[16.0pt] Course Learning Outcomes
[12.0pt] Upon completion of the course, students will be able to:
[12.0pt] 1. Perform Basic Logic Design
[12.0pt] 2. Understand the design of a simple computer system
[12.0pt] 3. Understand elements that make up a CPU (Central Processing

[12.0pt] 4. Design software with an understanding of how it will run on
[12.0pt] computers
[12.0pt] 5. Participate in discussions that involve impact of hardware and
[12.0pt] software elements on each other and the performance of the
[12.0pt] program
[12.0pt] 6. Take intermediate and advanced classes in computer architecture,
[12.0pt] systems, operating systems, cloud, parallel computing
