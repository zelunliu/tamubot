[16.0pt] Catalog Description
Programming Languages. (3-0). Credit 3. Exploration of the design space of
programming languages via an in-depth study of two programming languages, one
functional and one object-oriented; focuses on idiomatic uses of each language and
on features characteristic for each language. Prerequisite: Grade of C or better in
CSCE 221, or concurrent enrollment.
[16.0pt] Additional Course Details
This course explores the design space of programming languages via an in-depth
study of the most common features and constructs in programming languages,
using two programming languages to illustrate them: Haskell (Functional) and Java
(object-oriented).
The course focuses on idiomatic uses of functional and object-oriented languages
and on feature characteristics of those categories of languages. This course goes
into details of how programs written in high-level programming languages are
executed.
This includes topics such as static and dynamic environments, internal program
representations, type checking, and interpreting. Students will learn about the
implementation approaches of common abstraction mechanisms and modern
programming language constructs. The course explores the use of the functional
programming approach to design and implement programs, as well as study the
main features of object-oriented languages.
Although the course uses and studies the above specific programming languages,
learning these languages should be considered as a side benefit of the course, not
the main goal.

[16.0pt] Course Learning Outcomes
Upon completion of this course, students will be able to:
Use modern programming languages effectively
Develop the necessary skills to learn and adopt new programming languages
Evaluate the power of programming languages and their constructs
Apply language constructs, common abstraction mechanisms, and efficiency
considerations
Analyze the meaning and efficiency of functional and object-oriented
programs
Master many specific language concepts and use them effectively to build
correct and elegant software
Although the course uses and studies specific programming languages, learning
these languages should be considered as a side benefit of the course, not the main
goal.
