[16.0pt] Catalog Description
Computer Animation. (3-0). Credit 3. Investigation of computational problems in
computer animation; study of the mathematical and algorithmic foundations
behind various techniques used for computer animation for real-time and offline
use. Prerequisite: Grade of C or better in CSCE 315 or CSCE 331; grade of C or
better in or concurrent enrollment in CSCE 441.
[16.0pt] Course Prerequisites
Prerequisite/Corequisite(s):  Grade of C or better in CSCE 315 or CSCE 331; grade
of C or better in or concurrent enrollment in CSCE 441.
[16.0pt] Course Learning Outcomes
Upon completion of this course, the learner will be able to:
Make use of 3D rotations including quaternions
Apply key-frame interpolation of positions and orientations
Choose which spline curves to use for different applications based on
continuity requirements
Compute curve tangents and arc-length parameterization
Apply regression and curve fitting
Use Gaussian quadratures for definite integrals
Implement linear blend skinning
Compare and contrast linear blend skinning and dual quaternion skinning
Apply bilinear/trilinear interpolation for 2D/3D mesh deformation
Apply blend shapes for 3D facial animation
Implement a version of Simple Multi-Person Linear model
Implement gradient descent, Newton’s Method, and L-BFGS with backtracking
line search
Solve 2D inverse kinematics
Compare path-finding algorithms
Apply numerical integration schemes for ordinary differential equations
Explain the relationship between Position-Based Dynamics and Implicit Euler
Integration
Develop a cloth simulation system with collisions
