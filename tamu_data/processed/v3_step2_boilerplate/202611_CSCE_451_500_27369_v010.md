[16.0pt] Catalog Description
Software Reverse Engineering. (2-2). Credit 3. Overview of the compilation
mechanism to generate executable files and raw binary codes from source codes;
executable file formats for an operating system to run the binary code; disassembly
algorithms and control graph analysis; static and dynamic analyses; case studies on
code obfuscation, codebreaking, malware analysis. Prerequisite: Grade of C or
better in CSCE 313 or approval of instructor.
[16.0pt] Course Prerequisites
Prerequisite/Corequisite(s):  Grade of C or better in CSCE 313 or approval of
instructor.
[16.0pt] Course Learning Outcomes
Upon completion of this course, the learner will be able to:
[12.0pt] Analyze simple assembly programs to determine their functionality.
[12.0pt] Use a debugger to find code that has been packed or encrypted.
[12.0pt] Perform dynamic analysis to detect program behavior.
[12.0pt] Analyze malware safely in a contained environment.
[12.0pt] Understand process injection.
[12.0pt] Reverse engineer programs containing simple anti-disassembly and
[12.0pt] anti-debugging techniques.
