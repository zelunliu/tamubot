[16.0pt] Catalog Description
Advanced Networking and Security. (3-0). Credit 3. Security aspects of various
network protocols including investigation and tool development using "live"
machines and networks. Prerequisites: Graduate classification and approval of
instructor.
[16.0pt] Additional Course Details
[10.0pt] This course will introduce various modern topics in computer and network security. It will
[10.0pt] provide a thorough grounding in computer and network security suitable for those interested
[10.0pt] in conducting research in this area, as well as students more broadly interested in real-world
[10.0pt] security issues/techniques. Topics may span (but not limited to):
[10.0pt] Basic network & system security: e.g., software security, malware, botnet, intrusion
[10.0pt] detection
[10.0pt] Security in a software-defined world: e.g., SDN/NFV, Cloud, 5G/Edge, Vehicle security
[10.0pt] Mobile & IoT security: e.g., smartphone security, Alexa and voice app security
[10.0pt] Web and Web3 security: e.g., traditional web security, spam, Web3 (DeFi, blockchain)
[10.0pt] security
[10.0pt] AI/LLM security: e.g., LLM-based security, adversarial Learning / backdoor, advanced
[10.0pt] applications with AI
[16.0pt] Course Prerequisites
Prerequisite/Corequisite(s):  Graduate classification and approval of instructor.
[16.0pt] Course Learning Outcomes
Upon completion of this course, the learner will be able to:
[10.0pt] Analyze the complex trade-offs and architectural tensions inherent in securing modern
[10.0pt] distributed systems and networks.
[10.0pt] Synthesize fundamental security principles and mechanisms to design robust
[10.0pt] defenses against sophisticated threat models.
[10.0pt] Critique state-of-the-art security literature to identify open problems, methodological
[10.0pt] flaws, and emerging trends in the field.
[10.0pt] Demonstrate a practical working knowledge of offensive and/or defensive techniques
[10.0pt] to contextualize theoretical concepts.
