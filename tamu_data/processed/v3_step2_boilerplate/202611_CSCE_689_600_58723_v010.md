[16.0pt] Course Learning Outcomes
Upon completion of this course, the learner will be able to:
Evaluate properties of real and synthetic networks using several C++ and
Python software packages
Develop and evaluate graph machine learning models, including graph neural
networks and recommendation systems.
Create knowledge graphs from unstructured data and perform graph
embedding for link predictions in knowledge graphs.
Read research papers on graph machine learning and compare various state-
of-the-art methods
Develop a comprehensive project related to graph machine learning.
