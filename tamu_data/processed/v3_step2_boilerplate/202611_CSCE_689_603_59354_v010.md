[16.0pt] Additional Course Details
Algorithms for control-flow prediction including conditional branch prediction, indirect branch
prediction, return address prediction, and branch target buffer management. Addressing
technological constraints for implementation of branch prediction algorithms.
[16.0pt] Course Prerequisites
Prerequisite/Corequisite(s):  Approval of instructor.
[16.0pt] Course Learning Outcomes
Upon completion of this course, the learner will be able to:
• Explain and compare modern control-flow prediction algorithms
• Implement branch prediction in a detailed microarchitectural simulator
• Optimize branch predictors for a given class of workloads
